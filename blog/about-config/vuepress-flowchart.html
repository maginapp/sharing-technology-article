# 常用元素

  • start 开始
  • end 结束
  • operation 操作
  • condition 判断
  • inputoutput 输入输出
  • subroutine 子程序
flowchat
st=>start: 开始
e=>end: 结束
op=>operation: 我的操作
cond=>condition: 确认？
inout=>inputoutput: 输入
sub=>subroutine: 子程序

st->op->cond
cond(yes)->inout->sub->e
cond(no)->op
